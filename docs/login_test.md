# Login Module - Test Cases

| Test Case No. | Module Name | Test Scenario                                    | Test Description                                                                 | Expected Result                                                                  |
|---------------|-------------|--------------------------------------------------|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| **TC.1**      | Login       | Login With Valid Credentials                     | Enter correct email/username and password.                                         | User is logged in successfully.                                                   |
| **TC.2**      | Login       | Login With Registered Account                    | Attempt login using an existing active account.                                    | System authenticates and grants access.                                           |
| **TC.3**      | Login       | Login With Correct Case                          | Provide credentials with correct case sensitivity.                                 | System logs in the user.                                                          |
| **TC.4**      | Login       | Login After Trimming Spaces                       | Enter email/username and password with extra spaces accidentally.                  | System trims inputs and logs in user (if valid).                                  |
| **TC.5**      | Login       | Login With Valid Password Format                  | Provide password that matches stored password rules.                               | System logs in user.                                                               |
| **TC.6**      | Login       | Login Multiple Times Sequentially                 | Perform several login attempts with valid credentials.                             | System consistently logs in without issues.                                       |
| **TC.7**      | Login       | Login With Incorrect Password                     | Enter wrong password for an existing user.                                         | System denies login and shows credential error.                                   |
| **TC.8**      | Login       | Login With Unregistered Account                   | Try logging in with an account that does not exist.                                | System shows incorrect credentials message.                                       |
| **TC.9**      | Login       | Login Without Email/Username                      | Leave the username/email field blank.                                              | System prevents login and shows required field error.                             |
| **TC.10**     | Login       | Login Without Password                            | Try logging in without entering a password.                                        | System shows password required message.                                           |
| **TC.11**     | Login       | Login With Invalid Email Format                   | Enter improperly formatted email.                                                  | System shows invalid email format error.                                          |
| **TC.12**     | Login       | Rapid Repeated Failed Login Attempts   | Submit multiple incorrect passwords quickly.                                       | System may lock the account or continue showing invalid credential messages.       |
