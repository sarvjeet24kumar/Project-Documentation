# Functional Modules

## 1. Authentication Module
- JWT-based login and registration.
- Role-based access: Student / Admin.
- Password hashing and validation.


## 2. Event Module
- CRUD operations for events.
- Validation for event dates and locations.

## 3. Registration Module
- Students register/unregister for events.
- Prevent duplicate registrations.
- Track registration timestamps.

## 4. Dashboard Module
- Admin Dashboard: Upcoming events, participants.
- Student Dashboard: Upcoming events, Registered events.
